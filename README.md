# Curso Profesional de Javascript
Este es un curso de Platzi:
https://platzi.com/clases/javascript-profesional/

## Instrucciones

### Build mediaplayer package
1. `cd mediaplayer` 
2. `npm install --save`
3. `npm run build`

### Launch demo website 
1. `cd website` 
2. `npm install --save`
3. `npm start`
